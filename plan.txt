# Prompt: Build a Complaint Management System (Expo SDK 54 + MERN Stack)

## Project Overview

Build a full-stack Complaint Management System that allows users to submit complaints, track their status in real time, communicate with support staff, attach evidence, receive notifications, and rate the resolution process.

The system should consist of:

* Mobile App (Expo SDK 54 + React Native)
* Admin Web Portal (React)
* Backend API (Node.js + Express.js)
* MongoDB Database
* Socket.io for realtime updates
* JWT Authentication
* Role-Based Access Control (RBAC)

The application must be scalable, secure, responsive, production-ready, and follow clean architecture principles.
